# To-Do Application

A simple and user-friendly To-Do Application designed to help users organize, track, and manage their daily tasks efficiently. This application allows users to add, update, delete, and mark tasks as completed, improving productivity and time management.

## Features

➕ Add new tasks

✏️ Edit existing tasks

❌ Delete tasks

✔️ Mark tasks as completed

## Technology Stack

HTML, CSS, JavaScript

## How It Works

* User adds a task using the input form

* Task is displayed in the task list

* User can mark tasks as completed or delete them

* Changes are saved automatically
